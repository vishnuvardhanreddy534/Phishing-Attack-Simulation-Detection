# Task 11: Phishing Attack Simulation & Detection

## Objective
Simulate a phishing attack in a controlled environment using GoPhish to understand social engineering techniques and improve phishing detection and prevention awareness.

## Tools Used
- GoPhish
- Test SMTP Server (MailHog / Local SMTP)

## Project Structure
- `/report/Task_11_Phishing_Simulation_Report.pdf`
- `/screenshots/` (GoPhish dashboard images)
- `README.md`

## Simulation Steps
1. Setup GoPhish framework.
2. Create phishing email template.
3. Design fake landing page.
4. Launch phishing campaign on test users.
5. Analyze results via GoPhish dashboard.

## Learning Outcome
- Understanding phishing attack lifecycle
- Identifying phishing red flags
- Implementing effective prevention strategies

## Disclaimer
This project is for educational purposes only. No real users or credentials were harmed.
